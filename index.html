Useful CSS With Less Mixins

Snippets of reusable CSS by less mixins to develop faster and keep code readable

How to Use:

Download the UsefulCSS.less OR UsefulCSS.min.less and then copy the following code at the right place given here:

/* place it into your HTML <head> tag */
<link rel="stylesheet/less" href="UsefulCSS.min.less">

/* OR use @import in your less file */
@import "../UsefulCSS.min.less";

Note:

Load the less.min.js into your HTML <head> tag right before UsefulCSS.less to compile automatically the less file to CSS code, OR manually compile it with appropriate less compilers.

How it works:

As a rule, you can use the CSS properties you would expect just by adding a . to start them and putting arguments afterwards.

Whatever... let's say you want a elliptical border radius for an element, here you go:


Example:

div.radius{
	.border-radius(150px 150px 150px 150px, 75px 75px 75px 75px);
}

Output:

div.radius {
	-webkit-border-radius: 150px 150px 150px 150px / 75px 75px 75px 75px;
	-moz-border-radius: 150px 150px 150px 150px / 75px 75px 75px 75px;
	border-radius: 150px 150px 150px 150px / 75px 75px 75px 75px;
}

.font-face(@font-family-name; @font-name; @font-style; @font-weight; @woff; @woff2; @ttf; @eot; @svg);

Explain the variables:
  • @font-path Enter your font location.
    Default: "../fonts/@{font-family-name}";
  • @font-family-name Use this variable for the font-family property. [e.g: roboto]
  • @font-name Write the font name without the suffix. [e.g: roboto-bold]
  • @font-style normal | italic | oblique.
    Default: normal;
  • @font-weight normal | bold | bolder | lighter | number[e.g: 100,200,300,...,900].
    Default: normal;
  • @woff Set value "woff" to show woff font. IF set this value "all", shows all font suffix. e.g: woff,woff2,ttf and etc.
    Default: woff;
  • @woff2 Set value "woff2" to show woff2 font. IF you want to not show "woff2", set NULL;
    Default: NULL;
  • @ttf Set value "ttf" to show ttf font. IF you want to not show "ttf", set NULL;
    Default: NULL;
  • @eot Set value "eot" to show eot font. IF you want to not show "eot", set NULL;
    Default: NULL;
  • @svg Set value "svg" to show svg font. IF you want to not show "svg", set NULL;
    Default: NULL;

Note: @font-family-name variable is required and another variables is optional.


Fonts suffix and browsers support:
  • *.woff Pretty Modern Browsers [FF3.6+, IE9, Chrome6+, Saf5.1+]
  • *.woff2 Super Modern Browsers [FF39+,Chrome36+, Opera24+]
  • *.ttf Safari, Android, iOS
  • *.eot IE9 Compat Modes
  • *.eot?#iefix Only IE6-8
  • *.svg Old iOS

Explain the mixin:
  • .media-max() For special media size that needs you choose max-width.
  • .media-min() For special media size that needs you choose min-width.
  • .mobile() To make coding easier, for media size from the MaterializeCSS framework have been used.

  • .tablet() To make coding easier, for media size from the MaterializeCSS framework have been used.
    [min-width: 600px and max-width: 991.99px]
  • .desktop() To make coding easier, for media size from the MaterializeCSS framework have been used.
    [min-width: 992px and max-width: 1199.99px]
  • .desktop-wide() To make coding easier, for media size from the MaterializeCSS framework have been used
    [min-width: 1200px]
  • .mobile-small-bs() To make coding easier, for media size from the BootStrap V4.0 framework have been used
    [max-width: 576.99px]
  • .mobile-bs() To make coding easier, for media size from the BootStrap V4.0 framework have been used.
